# Moje Portfolio 🚀

### Opis projektu
Aplikacja Portfolio to moja osobista wizytówka w sieci, stworzona w celu prezentacji moich projektów, umiejętności technicznych oraz ścieżki zawodowej. Projekt został zaprojektowany z myślą o przejrzystości (UX) oraz pełnej responsywności (RWD).

### Główne Funkcjonalności
* **Dynamiczna Galeria Projektów:** Podgląd wybranych prac z linkami do kodu źródłowego (GitHub) oraz wersji demonstracyjnych (Live).
* **Sekcja "O mnie":** Krótka biografia oraz opis moich celów zawodowych.
* **Tech Stack:** Interaktywna lista technologii i narzędzi, którymi się posługuję.
* **Formularz Kontaktowy:** Umożliwia szybkie wysłanie wiadomości bezpośrednio z poziomu strony.
* **Dark/Light Mode:** Przełącznik motywu kolorystycznego dostosowany do preferencji użytkownika.
* **Responsywność:** Pełne wsparcie dla urządzeń mobilnych, tabletów i desktopów.
